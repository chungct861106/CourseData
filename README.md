## 目前觀察
- 文學院的外院人數、比例高(但比例逐年下降)
    => 語言必選修人數高，但國文自106年改為選修課程
- 理學院、社科院人數、比例高 => 外系必修人口數高
- 電資學院外院外系比例逐年增加 => 資工的外系修課人數成長快速
